# 完成済み機能

## マップ

- 150×150タイルのグリッドマップ（灰色）
- 黄色の外枠ライン（8px）
- 資源スポット（緑の菱形）を1200個ランダム配置（シード固定）

## カメラ

- WASD移動・マウス中ボタンドラッグ
- ホイールズーム
- マップ外へのスクロール制限なし（黄色線の外まで移動可能）

## コマンダー

- 青い正方形（P1）を左上エリアにスポーン
- クリックで選択、白枠で選択状態を表示
- 右クリックで移動
- マップ境界（黄色線）の外に出られない
- ホバーでハイライト表示

## 選択システム

- クリックで単体選択
- Shift+クリックで選択に追加
- ドラッグで矩形選択（ドラッグ閾値8px）
- Shift+ドラッグで選択に追加
- 地面クリックで選択解除
- コマンダー選択時のみ建設HUDが表示される

## CommanderHUD（底部UI）

- コマンダー選択時に画面下部に表示
- 建物ボタン8種（PAスタイル）
- アンロック済み：Metal Extractor、Factory
- ロック済み：Turret、Shield Gen、Radar、Wall、Adv Factory

## Metal Extractor 配置

- HUDのボタンを押すと配置モードに入る
- マウスに最寄りの資源スポットに吸い込み（Snap）
- 赤いリング = 既存Extractorに近すぎて置けない
- 左クリックで1個ずつ配置
- Shiftを押しながら配置すると配置モードを継続
- ドラッグで範囲選択→一括キュー追加（最近傍ルートで順番最適化）
- コマンダーが射程外の資源はクリックすると自動移動→到着後に建設開始（Move-and-Claim）
- 建設中は青いラインとキュー表示、3秒で完成
- 完成資源には青いリング表示
- 右クリックでキャンセル

## Factory 配置

- HUDのFactoryボタンを押すと配置モードに入る
- 左クリックで配置（建設キューに追加）
- Shiftを押しながら配置すると配置モードを継続
- マップ境界（黄色線）の外には配置不可（プレビューが赤くなる）
- 右クリックでキャンセル
- コマンダーが自動で建設位置へ移動し、10秒で完成

## Factory HUD

- Factoryをクリックすると画面下部にユニット生産HUDが表示
- ダブルクリックで全Factory選択 → 生産ボタンで全Factoryに同時キュー
- 生産キューとプログレスバー表示

## 衝突・押し出し

- コマンダーが資源スポットに重ならないようプッシュ処理あり

## ネットワーク

- WebSocket（ws://127.0.0.1:8080）で自動接続・3秒ごとに再接続試行
- コマンダー位置を50msごとにサーバーに送信
- 他プレイヤーのコマンダーを赤い正方形でリアルタイム表示
- プレイヤーの入退室を検知してスポーン／削除

## コードアーキテクチャ

- `scripts/managers/` にマネージャー分離（factory / resource / selection）
- `scripts/units/` にユニット種別分離（unit基底 / commander / infantry / tank）
- `scripts/factory_hud.gd` — Factory選択時のHUD
