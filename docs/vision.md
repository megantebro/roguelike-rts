# 完成品ビジョン

## コンセプト

PA:TITANSライクな 2D PVP RTS。
ファクトリーでユニットを量産して、相手のコマンダーを破壊したら勝ち。
普通のRTSと違うのは**ドラフトシステム**：一定数生産するたびに3択でユニットタイプを選び、毎回違う軍団でぶつかり合う。

---

## 勝利条件

- 相手の**コマンダー**を破壊したら勝ち
- マッチ後そのままリマッチ可能

---

## コアループ

1. コマンダーをスポーン、建設開始
2. ファクトリー・資源施設を建設
3. ファクトリーからユニットを連続生産
4. **ドラフトトリガー**：累計生産数が閾値に達すると3択カードが出る
5. 1枚選んでユニットタイプをアンロック → 生産ラインに追加
6. 軍団で相手コマンダーを撃破 → 勝利

---

## ユニット

| ユニット | 役割 |
|---------|------|
| コマンダー | 各1体。倒されたら即敗北。建設能力あり |
| 歩兵 | 安価・量産向き。最初からアンロック済み |
| 戦車 | 火力・耐久バランス型 |
| 砲兵 | 長射程・範囲攻撃。低機動 |
| 航空 | 地形無視・高速。奇襲向き |
| タイタン | ドラフト報酬のみ。超強力 |

## ドラフト閾値

| 回数 | 累計生産数 |
|------|-----------|
| 1回目 | 10体 |
| 2回目 | 25体 |
| 3回目以降 | +20体ごと |

---

## 資源システム

- **メタル**：資源スポットにMetal Extractorを建設して収入
- **エネルギー**：Power Generatorを建設して収入
- 両方を消費してユニット・施設を生産
