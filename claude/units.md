# ユニット仕様・設計

試作機2以降で実装するユニットの仕様と設計をまとめる。
コアコンセプト・ドラフトシステム・勝利条件は [../spec.md](../spec.md) を参照。

---

## 共通ステータス定義

全ユニットは以下のパラメータを持つ。値はゲームバランス調整で変動する前提。

| 項目 | 単位 | 説明 |
|------|------|------|
| `hp` | int | 最大HP。0で撃破 |
| `atk` | int | 1回の攻撃で与えるダメージ |
| `atk_range` | px | 攻撃可能距離（ユニット中心からの半径） |
| `atk_interval` | sec | 攻撃と攻撃の間隔（小さいほど高速） |
| `move_speed` | px/sec | 最大移動速度 |
| `sight` | px | 視界半径（敵自動索敵に使用） |
| `radius` | px | コリジョン半径（衝突・密度に使用） |
| `cost_metal` | int | 生産コスト（メタル） |
| `cost_energy` | int | 生産コスト（エネルギー） |
| `build_time` | sec | ファクトリーでの生産時間 |
| `targets` | enum | 攻撃可能対象 (`ground` / `air` / `both`) |
| `domain` | enum | 自身の所属 (`ground` / `air`) |

## コマンダー

各プレイヤーに1体だけ存在する司令官。倒されると敗北。

- **HP**: 5000
- **ATK**: 80 / 射程 250 / 攻撃間隔 1.0s
- **移動速度**: 80
- **特殊**: 建設能力あり（ファクトリー・資源施設を直接建設可能）
- **生産不可**：マッチ開始時に1体スポーンする固定ユニット

設計上の役割：序盤は唯一の建設ユニット、終盤は重戦力として前線にも出せる。
ただし倒されたら即敗北のため、最終的には後方で運用するのが定石。

## 歩兵（Infantry）

- **HP**: 100 / **ATK**: 15 / 射程 180 / 攻撃間隔 1.5/s
- **移動速度**: 100
- **コスト**: M30 / E10、生産時間 3秒
- **対地/対空**：両方OK

安価・短時間で量産可能な基本ユニット。1体ずつは弱いが数で押す。
ドラフト未到達でも最初からアンロックされている前提。

---

## 戦車（Tank）

- **HP**: 600 / **ATK**: 60 / 射程 220 / 攻撃間隔 0.8/s
- **移動速度**: 70
- **コスト**: M120 / E40、生産時間 8秒
- **対地のみ**

火力・耐久のバランス型。前線維持の主力。

---

## 砲兵（Artillery）

- **HP**: 250 / **ATK**: 120（範囲攻撃 半径60px） / 射程 500 / 攻撃間隔 0.3/s
- **移動速度**: 50
- **コスト**: M200 / E80、生産時間 12秒
- **対地のみ**

長射程・低機動の砲撃ユニット。建物の遠距離撃ち抜きや密集軍団への範囲攻撃が役割。
近接戦闘には弱いため歩兵・戦車での護衛が必須。

---

## 航空（Air）

- **HP**: 200 / **ATK**: 30 / 射程 200 / 攻撃間隔 1.2/s
- **移動速度**: 150
- **コスト**: M150 / E100、生産時間 10秒
- **両方攻撃可** / 自身は`air`ドメイン

地形無視で高速移動。奇襲・後方資源の破壊が役割。
対空攻撃手段（歩兵・航空）が無い相手には致命的に刺さる。

## 攻撃ルールの共通仕様

- 索敵：`sight` 半径内に敵が入ると自動で接近・攻撃を開始（停止指示中は除く）
- 命中：射程内なら必中（ドッジ・命中率なし）
- ダメージ：単発は固定値。範囲攻撃（AoE）は中心からの距離で減衰なし（簡略化）
- 攻撃中の移動：射程外に出ると自動追従、`atk_interval` の途中で中断可
- 死亡：HPが0以下で即座に消滅。死体・経験値は無し

## 移動ルールの共通仕様

- 経路：A*等のグリッド経路探索（試作機2で簡易実装、後で改善）
- 衝突：`radius` ベースで他ユニットを押し退ける（既存の `bar_overlay` / `unit.gd` の押し戻しロジックを流用）
- 集団移動：複数選択時は目標点を中心に円形オフセット（既存挙動を踏襲）
- 航空ユニット：障害物・他地上ユニットを無視、航空同士のみ衝突判定
