# roguelike-rts 仕様書

## コンセプト

PA:TITANSライクな大規模2D PVP RTS。
ファクトリーでユニットを量産し、相手のコマンダーを破壊することが目標。
通常RTSとの違いは**ドラフトシステム**：一定数生産するたびに3択でユニットタイプを選び、
毎回異なる軍団構成でぶつかり合う。

---

## 勝利条件

- 相手の**コマンダー**を破壊したプレイヤーの勝利
- マッチ終了後そのままリマッチ可能

---

## コアループ

1. コマンダーをスポーン、拠点エリアに配置してマッチ開始
2. 建設ユニットがファクトリー・資源施設を建設
3. ファクトリーからユニットを連続生産
4. **ドラフトトリガー**：累計生産数が閾値に達すると3択カードが提示される
5. 1枚選び新ユニットタイプをアンロック → 生産ラインに追加
6. 軍団を指揮して相手コマンダーを撃破 → 勝利

---

## ユニット設計

### コマンダー
- 各プレイヤーに1体のみ
- 建設能力を持つ（ファクトリー・施設を自分でも建てられる）
- 戦闘力は高いが倒されたら即敗北

### 通常ユニット
| 役割 | 特徴 |
|------|------|
| 歩兵 | 安価・大量生産向き |
| 戦車 | 火力・耐久バランス型 |
| 砲兵 | 長射程・低機動 |
| 航空 | 高速・地形無視 |
| タイタン | レア超強力ユニット、ドラフト報酬のみ入手可 |

### ドラフトシステム
- 累計生産数が閾値に達するたびに発動
- 3種のユニットカードをランダム提示 → 1枚選択してアンロック
- 以後そのタイプをファクトリーで生産可能になる

| 回数 | 累計生産数 |
|------|-----------|
| 1回目 | 10体 |
| 2回目 | 25体 |
| 3回目以降 | +20体ごと |

---

## 資源システム

- **メタル**：採掘施設をマップ上の資源点に建設して収入
- **エネルギー**：発電施設を建設して収入
- 両方を消費してユニット・施設を生産
- 建設ユニットが施設を建て、コマンダーも建設可能

---

## マップ設計

- 2Dトップダウン、左右対称マップ（PVP用）
- 中央に資源点が集中 → 序盤の奪い合いが発生
- 地形：平地 / 障害物（壁・水）でルート制限
